# Hacker-Style Portfolio

A bespoke, highly interactive, cyberpunk-themed portfolio tailored for a Backend Developer & NLP/AI Researcher. Built with modern web technologies to be fast, scalable, and visually impressive.

## Technology Stack

- **Framework**: React.js (via Vite)
- **Styling**: Vanilla CSS (CSS Variables, Flexbox/Grid, Keyframe Animations)
- **Typography**: Fira Code (Monospace) & Inter (Sans Serif)

## Project Structure

The project has been migrated from a legacy Hugo template to a modern Vite SPA. The structure is as follows:

```
├── public/                 # Static assets (favicons, etc.)
├── src/
│   ├── components/         # Reusable React components
│   │   ├── HeroTerminal.jsx # Animated typing header
│   │   ├── Lists.jsx        # Components for rendering Experience/Education/Publications
│   │   ├── SectionViewer.jsx# Hacker-window styled container wrapper
│   │   └── SkillMatrix.jsx  # Visual progress-bar component for skills
│   ├── data/
│   │   └── resumeData.js    # JSON/Object structured resume data (EASILY EXTENSIBLE)
│   ├── App.jsx             # Main layout orchestration
│   ├── index.css           # Global theme variables, animations, and typography
│   └── main.jsx            # React entrypoint
├── old_hugo_backup/        # Backup of the legacy Hugo template files
├── index.html              # HTML entrypoint
├── vite.config.js          # Vite configuration
└── package.json            # Dependencies and scripts
```

## How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:5173 to view it in your browser.

## How to Update Content (Scalability)

This project has been explicitly designed for easy updates without touching the UI component logic.

To add a new award, publication, or work experience, simply open `src/data/resumeData.js` and add an object to the related array. 

For example, to add an award:
```javascript
export const awards = [
  // ... existing awards
  { title: "New AI Hackathon Winner", year: "2026" }
];
```
The React components automatically map over these arrays and render them via the hacker styling.

## Deployment

To deploy to GitHub Pages:
1. Ensure the `base: './'` is set in `vite.config.js` (already configured).
2. Run the build script:
   ```bash
   npm run build
   ```
3. Upload/Deploy the generated `dist` folder to your `gh-pages` branch or configure a GitHub Action to deploy it automatically!
